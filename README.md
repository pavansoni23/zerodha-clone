# Zerodha-clone

## Overview
Zerodha-clone is an investment application inspired by Zerodha, developed using the MERN stack and React.
This project extends the Kite Dashboard to provide users with comprehensive tools for managing and analyzing their investments.

## Features
- **Kite Dashboard:** 
  - View and manage your watchlist.
  - Access detailed information on holdings, positions, and funds.

- **Investment Analytics:** 
  - Utilize interactive charts powered by Chart.js for in-depth analysis of investment data.

## Technologies Used
- **Frontend:** React
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Data Visualization:** Chart.js

## Key Contributions
- Developed a dynamic dashboard for real-time management of investment data.
- Integrated interactive charts to offer detailed visual analytics of investments.
- Ensured seamless integration between frontend and backend services for an optimal user experience.
